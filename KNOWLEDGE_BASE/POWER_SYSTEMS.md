# Power Systems Knowledge

## 0. Power设计原则
- IT ZONE自带UPS，提供5-8分钟切换

### BESS 连接方式
Grid → BESS → IT Zone → IT


### 柴油发电机连接方式
Grid → ATS → IT Zone → IT
        ↑
      Generator



## 1. UPS选型逻辑
- UPS容量 ≥ IT Load / PF × 1.2
- UPS内置于IT ZONE的设备内 （AC40或者DC40）提供一站式的不间断电服务
- 现在采用的产品是EATON 9395XR UPS，已经具备UL认证。知识点位于: /Users/yurimeng/.openclaw/workspace-engineer/KNOWLEDGE_BASE/UPS_EATON_9395XR.md

## 6. BESS和柴油发电机的对比

### 6.1 系统级对比
| 对比维度 | BESS 方案（Transformer → Switchgear → BESS → AC40(内置UPS)） | 柴油发电机方案（Grid + ATS + Generator） |
|-----------|-------------------------------------------------------------|------------------------------------------|
| 系统架构位置 | 串联在主供电路径 | 并联备用电源 |
| 电源类型 | 电子电源（PCS逆变） | 机械发电（柴油发动机） |
| 切换时间 | <10ms（电子级） | 8–30秒（机械启动） |
| 是否无缝供电 | 是（AC40 UPS覆盖PCS瞬态） | 否（依赖UPS桥接） |
| 电能质量改善 | 可稳压、滤波、无功补偿 | 无电能质量优化 |
| 频率稳定性 | 高频控制，稳定性高 | 受机械转速影响 |
| 电压波动抑制 | 主动抑制 | 无能力 |
| 孤岛运行能力 | 支持 | 支持 |
| 持续供电时间 | 1–4小时（典型） | 24小时以上（取决于燃料） |
| 长时间运行能力 | 有限 | 理论无限（补油即可） |
| 初始投资（CAPEX） | 高 | 较低 |
| 运行成本（OPEX） | 低（无燃料） | 高（燃料+维护） |
| 维护复杂度 | 低（电子系统） | 高（机械维护） |
| 噪音 | 无 | 高 |
| 排放 | 无排放 | 高排放 |
| 城市部署适配性 | 极高 | 受环保限制 |
| 碳排放影响 | 低 | 高 |
| 电网互动能力 | 可削峰填谷、套利 | 无 |
| ESG友好度 | 高 | 低 |
| 单点风险 | PCS关键点（建议N+1） | ATS或发动机风险 |
| 扩展性 | 模块化扩容 | 扩展复杂 |
| 并网收益可能性 | 有 | 无 |
| 极端长时间停电 | 受电池容量限制 | 更适合 |
| 审批难度 | 低 | 高（排放+储油） |
| Edge AI适配性 | 非常适合 | 次优 |

### 6.2 系统层行为对比
| 层级 | BESS 架构行为 | 柴油机架构行为 |
|------|---------------|---------------|
| 一级供电 | BESS主供电 | Grid主供电 |
| 失电瞬间 | PCS持续供电 | ATS切换 + 发电机启动 |
| AC40 UPS作用 | 覆盖PCS瞬态 | 覆盖启动时间 |
| 电网波动 | 主动修正 | 直接传导至UPS |
| MDC扩展 | 线性扩展储能模块 | 增加大型发电机 |

### 6.3 工程决策维度
| 类别 | BESS优势 | BESS劣势 |
|------|------------|------------|
| 技术层面 | 毫秒级切换、稳压能力强 | 供电时间有限 |
| 运维层面 | 无机械维护 | 电池衰减 |
| 经济层面 | 可参与电网收益 | 初期投资高 |
| 战略层面 | ESG友好、城市适配 | 容量需精确规划 |

| 类别 | 柴油机优势 | 柴油机劣势 |
|------|------------|------------|
| 技术层面 | 长时间供电 | 启动慢 |
| 运维层面 | 技术成熟 | 维护复杂 |
| 经济层面 | 设备便宜 | 燃料成本高 |
| 战略层面 | 偏远地区适用 | 不适合城市 |


### 6.4 场景推荐

| 场景 | 推荐方案 |
|------|----------|
| 城市边缘AI | BESS |
| 高ESG要求 | BESS |
| 电网稳定但波动 | BESS |
| >8小时 автоном供电 | 柴油机 |
| 偏远矿区 | 柴油机 |
| 极端高可靠需求 | BESS + 小型柴油 |
